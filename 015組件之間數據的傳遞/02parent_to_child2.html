父組件將值傳遞到子組件的 model 的 props 後就可以放到子組件 model 的 template



在視圖的子組件裡寫子組件的 model 的 props 屬性


在視圖的子組件裡 v-bind 子組件的 model 的 props 屬性

v-bind 裡面只能寫變數，這裡在屬性值裡使用單引號表示變數是字串


在視圖的子組件裡 v-bind 子組件的 model 的 props 屬性

視圖裡的 v-bind:props 屬性值裡放父組件 model 裡的 data

就可以將父組件 model 的 data 傳遞到子組件 model 的 props


視圖子組件 v-for + 父組件傳值到子組件

視圖子組件 v-for 迭代父組件 model 的 data

視圖子組件 v-bind 子組件 model 的 props，屬性值是 v-for 迭代出來的 item
